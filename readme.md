  <h2> 2. Le jeu : </h2>

    Ce jeu se joue à 2 joueurs : 
    
    Alignez les X ou les O avant votre adversaire ! 

  >             1   2   3
  >
  >           ┌───┬───┬───┐
  >         A │ O │ O │   │          O : Julien
  >           ├───┼───┼───┤
  >         B │ X │ X │   │          X : Kevin
  >           ├───┼───┼───┤
  >         C │   │   │   │
  >           └───┴───┴───┘
  >
